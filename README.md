## Description

This is the working code for a TreeHacks 2023 React Demo.

<br>

# Steps to get up and Running

### 1. Clone the repo

`git clone https://github.com/ypat125/THordle.git`

### 2. Install Dependencies

```
$ cd Thordle
# npm install
```

### 3. Run the Project!

```
npm start
```
